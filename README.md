# Donaton Frontend - Interfaz de Usuario

Este repositorio contiene la aplicación cliente de la plataforma Donaton, diseñada para recibir donaciones ciudadanas y proporcionar un panel de administración para la gestión de recursos logísticos.

## Arquitectura y Tecnologías
- **Framework:** React + Vite
- **Navegación:** React Router DOM
- **Arquitectura de conexión:** Consume exclusivamente al BFF en el puerto 8080.

## Requisitos Previos
- Node.js y NPM instalados.

## Instalación y Ejecución en Desarrollo
1. Clonar el repositorio.
2. Instalar las dependencias de NPM:
   ```bash
   npm install
3. Levantar el servidor de desarrollo:
   ```bash
   npm run dev
4. Acceder en el navegador a http://localhost:5173.
